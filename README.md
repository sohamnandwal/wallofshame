# Wall of Shame
A global wall of shame where developers can post snippets of code they are embarrassed by or that went horribly wrong. Users can post notes anonymously and also comment on other's notes.

---
## Installation & How to Run
This is a simple React JS + vite frontend with an Express JS backend server with MongoDB for datastorage.
To run the site,
1. Clone the repository in your local machine.
2. Ensure that node and npm are installed. After that, install the package.json dependencies using ```npm install``` in each, backend and frontend folder
3. Also install local Mongodb server from this website [MongoDB](https://www.mongodb.com/try/download/community)
4. Now, from the root directory, change directory (cd) into frontend and run ``` npm start``` to start the backend and in another terminal, run ```npm run dev``` to run the site in development mode.
5. Ta-da ! The site is running !
